# 🌍 Life Expectancy Prediction using Linear Regression

## 📌 Project Overview  
This project uses **Linear Regression** to predict life expectancy based on various health and economic factors. The dataset includes features like adult mortality, alcohol consumption, GDP, schooling, and HIV/AIDS prevalence.

The goal is to analyze how these factors influence life expectancy and build a predictive model.

---

## 🛠️ Technologies Used  
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Scikit-learn  

---

## 📂 Dataset  
- File: `Life Expectancy Data.csv`  
- Contains country-wise health and economic data  

---

## ⚙️ Features Used  
- Adult Mortality  
- Alcohol  
- GDP  
- Schooling  
- HIV/AIDS  

---

## 🚀 How It Works  
1. Load dataset using Pandas  
2. Remove missing values  
3. Select relevant features  
4. Split data into training and testing sets  
5. Train Linear Regression model  
6. Predict life expectancy  
7. Visualize results  
8. Evaluate model performance  

---

## 📊 Model Evaluation  
- **Mean Squared Error (MSE)**  
- **R² Score**  
- **Coefficients**  
- **Intercept**  

---

## 📈 Output  
- Displays dataset preview  
- Prints MSE and R² score  
- Shows model coefficients  
- Scatter plot of Actual vs Predicted values
